Данный сайт написан на фреймворке Django языка Python. Это демонстрационный сайт интернет-магазина.
В нем присутствуют категории, подкатегории и товары с возможностью добавления их в избранное и корзину.
Также реализована возможность регистрации, смены данных профиля, смена и восстановления пароля (при восстановлении
пароля ссылка для смены пароля придет на указанную при регистрации почту с почтового сервиса yandex.ru). При
входе в учетную запись товары в гостевом избранном и гостевой корзине сохраняются.
Реализована полноценная админ-панель сайта. Имеется пагинация (см. подкатегорию смартфоны и гаджеты)


# Установка и запуск приложения
1. Клонирование репозитория:
`git clone https://github.com/SibAlive/dns-django.git`
`cd dns-django`
2. Создание виртуального окружения (для Windows):
`python -m venv venv`
`venv/Scripts/activate`
4. Настройка переменных окружения (файл .env необходимо заполнить своими данными:
`cp .env_example .env`
5. Открыть терминал из папки с проектом
6. Запустить контейнер
`docker compose build`
`docker compose up -d`
7. Восстановления дампа БД
`docker cp backup django_postgres:/tmp/backup.sql`
`docker exec -i django_postgres psql -U your_user -d your_database -f /tmp/backup.sql`
8. Перейти в браузер по ссылке:
http://localhost:8000/
